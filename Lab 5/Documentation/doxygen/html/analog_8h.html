Detailed Description

Routines for setting up and reading from the ADC.

This contains the functions for reading analog values from the LTC1859 ADC on the TWR-ADCDAC-LTC board. The ADC is 16-bit, and configured with a bipolar voltage range of +/- 10V. This contains the functions for writing analog values to the LTC2704 DAC on the TWR-ADCDAC-LTC board. The DAC is 16-bit, and configured with a bipolar voltage range of +/- 10V.
